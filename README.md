## Installation

M7CarouselView is available through [M2Spec](https://github.com/hellochenms/M2Spec). To install
it, simply add the following line to your Podfile:

```ruby
source 'git@github.com:hellochenms/M2Spec.git'

pod 'M7CarouselView'
```
